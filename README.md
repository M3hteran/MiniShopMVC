# 🛒 MiniShopMVC

MiniShopMVC, **ASP.NET Core MVC** ile geliştirilmiş katmanlı mimariye sahip bir mini e-ticaret uygulamasıdır.

Bu proje, ASP.NET Core MVC, Entity Framework Core ve N-Tier Architecture konularında kendimi geliştirmek amacıyla hazırlanmıştır.

---

# 🚀 Kullanılan Teknolojiler

- ASP.NET Core MVC
- Entity Framework Core
- SQL Server
- LINQ
- Bootstrap 5
- HTML5
- CSS3
- JavaScript
- Identity Authentication
- Role Based Authorization

---

# 🏗️ Mimari

Proje katmanlı mimari kullanılarak geliştirilmiştir.

```
MiniShopMVC
│
├── Controllers
├── Models
├── ViewModels
├── Views
├── Services
├── Repositories
├── Data
├── Migrations
└── wwwroot
```

Projede;

- Generic Repository Pattern
- Service Layer Pattern
- Dependency Injection
- Repository Pattern

kullanılmıştır.

# ⚙️ Kurulum

```bash
git clone https://github.com/KULLANICI_ADIN/MiniShopMVC.git
```

```bash
cd MiniShopMVC
```

Connection String'i kendi SQL Server bilgilerinize göre düzenleyin.

Daha sonra:

```bash
Update-Database
```

ve projeyi çalıştırın.
